# Api Statements

| Method | Path             | Description                       |
| :----- | :--------------- | :-------------------------------- |
| GET    | /post            | 전체 게시글 조회                  |
| POST   | /post            | 게시글 작성                       |
| PUT    | /post/:id        | 게시글 수정                       |
| DELETE | /post/:id        | 게시글 삭제                       |
| GET    | /post/:id        | 게시글 조회                       |
| GET    | /comment         | 댓글 조회                         | 
| POST   | /comment         | 댓글 작성                         |
| PUT    | /comment/:id     | 댓글 수정                         |
| DELETE | /comment/:id     | 댓글 삭제                         |
